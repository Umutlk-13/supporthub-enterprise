## Project Description

SupportHub Enterprise v2.0 is a web-based IT Support and Help Desk Management System developed with Flask.

## Main Features

- Role-based login system
- Secure password hashing
- Admin user management
- Customer ticket creation
- Operation supervisor assignment
- Technician task list
- Expert technician role
- Ticket comments
- Ticket timeline
- Notification system
- Dark / Light theme button
- Excel export
- PDF export
- Inventory asset management
- Stock management
- Profile page
- Change password interface
- SQLite database
